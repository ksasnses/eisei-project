## 1. このアプリでできること

- **今日やることを自動で決めてくれる**  
  生活スケジュール・部活の有無・試験日をもとに、1日ごとの学習計画を作成します。

- **復習を忘れない**  
  忘却曲線に基づき、やった内容を復習するタイミングを自動で計算します。

- **勉強時間の管理**  
  ポモドーロ法（25〜30分集中＋短い休憩）で学習を進められます。

- **共通テストの本番練習**  
  共通テストと同じ時間割で「仮本番」ができます。

## 2. 初めて使うとき（最初の設定）

初回起動時、「設定ウィザード」が始まります。次の順番で入力していきます。

### ステップ1：志望校タイプを選ぶ

- 「国公立文系」「国公立理系」「私立文系」などから、自分に合うものを選びます。
- 共通テストの受験科目が自動で決まります。

### ステップ2：受験科目を選ぶ

- 志望校タイプに合わせて、必要な科目を選びます。
- 例：理科基礎なら「物理基礎・化学基礎・生物基礎・地学基礎」のうち2つを選択、など。
- 選択し終えたら「次へ」を押します。

### ステップ3：各科目の現在の点数と目標を入力

- 各科目の「現在の点数」と「目標点数」を入力します。
- 「苦手度」も選ぶと、計画がより自分に合ったものになります。

### ステップ4：試験日と勉強開始日

- **試験日**：共通テスト本番の日付を入力します。
- **勉強開始日**：いつから計画を始めるかを選びます。今日から始めるなら今日の日付でOKです。

### ステップ5：生活スケジュール

- **起床時間・就寝時間**
- **学校の始業・終業時間**
- **通学時間（片道）**
- **食事・風呂の時間**
- **自由時間のバッファ**：予定外の用事に備える時間（目安：30分）

### ステップ6：部活スケジュール

- **部活がある曜日**：月・火・水…のうち、部活がある曜日をタップして選びます。
- **平日の部活時間**：開始・終了時刻
- **土日の部活時間**：土日に部活がある場合

※ 平日は部活があると時間が少ないため、**英語・数学のみ**が計画されます。土日や部活のない日は国語・理科・社会なども含まれます。

### ステップ7：夏休み期間（任意）

- 夏休みの開始日・終了日を入力すると、夏休み期間は別のスケジュールになります。

### 完了

「学習計画を開始」を押すと、設定が保存され、ホーム画面が表示されます。

## 3. 画面の見方と使い方

### 画面下部の5つのタブ

| タブ | 内容 |
|------|------|
| **ホーム** | 今日の学習計画、タスク一覧、達成状況 |
| **カレンダー** | 週ごとの計画、イベント登録（試合・学校行事など） |
| **タイマー** | ポモドーロタイマー、仮本番モード |
| **科目** | 受験科目一覧、各科目の詳細へ |
| **設定** | プロフィール、スケジュール、学習ルールの変更 |

---

### ホーム画面

- **試験まであと○日**：試験日までの残り日数
- **フェーズ**：基礎期・実践期・直前期（試験までの日数で自動で切り替わります）
- **今日の学習計画**：その日にやる科目とタスクの一覧
- **勉強可能時間**：その日に使える勉強時間
- **時間配分**：勉強可能時間・ゆとり・タスク合計の内訳

**タスクの進め方**

1. 各タスクの「開始」を押すと、タイマー画面に移動します。
2. 終わったら「完了」を押すと、記録されます。
3. 1ブロック分まとめて終わった場合は「ブロック完了」で一括登録できます。

---

### カレンダー画面

- **週の切り替え**：左右の矢印で前週・翌週に移動
- **各日の表示**：その日の科目ブロック（英語・数学・国語など）が色付きで表示されます
- **イベント登録**：「＋」ボタンで試合日・学校行事・模試などを登録できます
  - 試合日：テニスなど部活の試合
  - 学校行事：文化祭、修学旅行など
  - 模試：共通テスト模試など  
  登録した日は「試合日」「イベント日」として、計画が自動で軽めになります。

---

### 科目画面

- 受験科目の一覧が表示されます。
- 各科目をタップすると「科目詳細」画面に移動します。
  - 現在の点数・目標点
  - 登録した教材と進捗
  - 得点推移のグラフ
  - 復習の進み具合

---

### 設定画面

- **得点戦略シミュレーター**：現在→目標のギャップを分析
- **プロフィール編集**：名前、志望校タイプ、試験日、勉強開始日
- **1日の生活スケジュール**：起床・就寝・学校時間など
- **部活スケジュール**：部活の曜日と時間
- **学習ルール設定**：曜日別のスケジュール、復習ルールなどを細かく変更（上級者向け）
- **登録教材一覧**：全科目の教材と進捗
- **データのバックアップ**：エクスポート／インポート
- **データのリセット**：すべてのデータを削除して最初からやり直す

## 4. 平日・土日・夏休みの学習内容

アプリでは「その日の種類」に応じて、取り組む科目が自動で決まります。

| 日付の種類 | 主な科目 |
|------------|----------|
| **平日・部活あり** | 英語・数学のみ |
| **平日・部活なし** | 英語・数学のみ |
| **土日・休日** | 英語・数学・国語・理科・社会 |
| **夏休み・部活あり** | 英語・数学 |
| **夏休み・部活なし** | 英語・数学・国語・理科・社会・情報 |
| **試合日** | 暗記確認のみ（軽め） |
| **イベント日** | 復習のみ（軽め） |

※ 勉強可能時間が足りない日は、優先度の低い科目から自動で調整されます。

## 5. 教材の登録と進捗管理

### 教材を登録する

1. **科目**タブ → 登録したい科目をタップ
2. 「教材を追加」ボタンを押す
3. **自分で入力**または**テンプレート**のどちらかで登録
   - **自分で入力**：教材名、章数、1章あたりの時間などを入力
   - **テンプレート**：人気の参考書・問題集の候補から選べます

### 教材の進捗

- タスクを「完了」にすると、教材の進捗が自動で進みます。
- 科目詳細画面で、何章まで進んだか・残り何章かを確認できます。
- **一時停止**：今は使っていない教材は一時停止にすると、計画から除外されます。
- **入れ替え**：新しい教材に変えたときは「入れ替え」で古いものを終了し、新しいものを登録できます。

## 6. ポモドーロタイマーの使い方

### 通常のポモドーロ

1. ホーム画面でタスクの「開始」を押すと、タイマー画面に移動します。
2. タイマーがスタートします（暗記型は約20分、思考型は約30分など、タスクの種類で変わります）。
3. 終わったら「完了」を押して記録します。
4. 休憩をはさみながら、次のタスクに進みます。

### 仮本番モード（共通テスト本番練習）

1. **タイマー**タブを開く
2. 「仮本番モード」を選ぶ
3. 1日目・2日目を選び、共通テストと同じ時間割でタイマーが動きます。
4. 各科目の終了時に点数を入力すると、記録されます。

※ 直前期（試験30日以内）は、仮本番を月2回くらいのペースでやると効果的です。

## 7. よくある質問

### Q1. 計画がうまく表示されない

- 設定画面の「この目標でスケジュールを再計算」ボタンを押してみてください。
- 生活スケジュールや部活の曜日が正しく入力されているか確認してください。

### Q2. 平日なのに国語・理科・社会が入っている

- アプリを一度再読み込み（ページを更新）すると、平日は英語・数学のみの計画に更新されることがあります。
- 設定画面の「1日の勉強可能時間」で表示されている曜日別の内容と、ホーム画面の計画が一致するように設計されています。

### Q3. 教材を変えたい

- 科目詳細画面で、その教材のメニュー（⋮）から「入れ替え」を選び、新しい教材を登録します。

### Q4. データを消して最初からやり直したい

- 設定画面の一番下にある「データのリセット」から実行できます。すべて消えて、最初の設定画面に戻ります。

### Q5. スマホで使える？

- PWA（プログレッシブ・Webアプリ）対応なので、スマホのブラウザで開いて、ホーム画面に追加すればアプリのように使えます。

## おわりに

このアプリは、**生活リズムに合わせて無理のない計画**を自動で作ることを目指しています。  
部活がある日は英語・数学に絞り、土日や夏休みは国語・理科・社会も含めてバランスよく勉強できるように設計されています。

分からないことがあれば、保護者の方や学校の先生に聞いてみてください。  
頑張ってください。
